# Excel Sales Dashboard Analysis

## Overview
This project contains an interactive Excel dashboard built to analyze
sales performance of an e-commerce store for the year 2022.

The dashboard provides insights into sales trends, customer behavior,
order status, top-performing states, and sales channels.

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts and Slicers
- Data Cleaning

---

## Key Observations
- Women customers contribute around 64% of total sales.
- Around 92% of orders were successfully delivered.
- Maharashtra is the top-performing state by sales.
- Amazon is the leading sales channel.
- Sales peak in the month of March.
